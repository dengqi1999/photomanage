项目是一个简单的照片管理网站，现在实现的功能有照片的上传、修改和删除（包括批量删除）
项目主要使用了基于Spinrg+SpringMvc+Mybatis+Bootstrap。使用Spinrg+SpringMvc+Mybatis来搭建服务器端，使用Bootstrap来搭建前端页面。
